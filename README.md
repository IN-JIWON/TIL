### 리눅스 커맨드라인 기초
```
1. pwd // print working directory(현 작업물 저장 경로)
```
```
2. cd // change directory(경로 이동)
```
> cd ../(상위 폴더로 이동)
```
3. ls // list
```
> -a(숨김파일)<br>
> -l(자세한 목록)<br>
> -al(숨김파일 + 자세한 목록)
```
4. vim // 새로운 파일 만들기
```
> i(입력모드 전환)  
> Esc(명령모드로 전환)  
> 명령어(w:저장, q:나가기, wq:저장 후 나가기)
```
* clear: 창 깨끗하게
* history: 이전에 썼던 함수들 리스트로 보여줌
* ↑: 이전에 썼던 함수들
